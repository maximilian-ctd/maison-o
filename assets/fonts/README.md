# Schriftarten - Fonts

## Anantason Ultra

Die Schriftart "Anantason Ultra" wird für das Logo "MAISON Ô" verwendet.

### Schriftart hinzufügen

1. **Schriftart-Dateien hochladen:**
   - Lade die Schriftart-Dateien in diesen Ordner hoch:
     - `Anantason-Ultra.woff2` (empfohlen)
     - `Anantason-Ultra.woff` (Fallback)
     - `Anantason-Ultra.ttf` (Fallback)

2. **Dateinamen:**
   - Stelle sicher, dass die Dateinamen genau so lauten:
     - `Anantason-Ultra.woff2`
     - `Anantason-Ultra.woff`
     - `Anantason-Ultra.ttf`

3. **Shopify Assets:**
   - In Shopify: Gehe zu Theme → Assets
   - Lade die Schriftart-Dateien hoch
   - Die Schriftart wird automatisch geladen

### Schriftart erwerben

Die Schriftart "Anantason Ultra" ist eine kommerzielle Schriftart von Jipatype.
- Verfügbar auf: [MyFonts](https://www.myfonts.com)
- Stelle sicher, dass du eine gültige Lizenz für die Web-Nutzung hast

### Alternative: Lokale Schriftart-Dateien

Falls du die Schriftart-Dateien bereits hast:
1. Kopiere sie in `assets/fonts/`
2. Die Schriftart wird automatisch geladen

### Fallback

Falls die Schriftart nicht geladen werden kann, wird automatisch die Standard-Schriftart verwendet.
