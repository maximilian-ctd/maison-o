# Moon Dance Schriftart Installation

Die Schriftart "Moon Dance" wird für die Signatur "Noa & Maximilian" verwendet.

## Schriftart-Dateien hinzufügen

Bitte fügen Sie die folgenden Schriftart-Dateien zum `assets/fonts/` Ordner hinzu:

- `MoonDance-Regular.woff2` (empfohlen)
- `MoonDance-Regular.woff` (Fallback)
- `MoonDance-Regular.ttf` (Fallback)

## Download-Quellen

Die Schriftart kann von folgenden Quellen heruntergeladen werden:

1. **GitHub (Google Fonts)**: https://github.com/googlefonts/moondance
2. **MadFonts**: https://madfonts.com/moon-dance-font/
3. **FontMeme**: https://fontmeme.com/schriftarten/moon-dance-schriftart/

## Lizenz

Die Schriftart ist unter der SIL Open Font License (OFL) lizenziert und kann kostenlos für private und kommerzielle Zwecke verwendet werden.

## Nach dem Hinzufügen

Nachdem Sie die Schriftart-Dateien hinzugefügt haben, wird die Schriftart automatisch für die Signatur verwendet.
